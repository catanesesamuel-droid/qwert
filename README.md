# Practica-SSDLC

En esta practica se tienen tres grandes partes, la parte del frontend, la parte de nginx que se encarga de tener comunicacion https y el backend que en este caso se llama programacion segura skeleton.
Ademas de ello esta el docker compose el cual va a levantar todo el servicio y luego una vez arriba se debe introducir esto: https://localhost:8443 en cualquier buscador.
Por otro lado para iniciar sesion como admin se debe iniciar con USERNAME:admin  PASSWORD:admin123 para asi poder revisar las funcionalidades.

El script proporcionado se puede usar en linux para levantar el sistema, se descomprime el zip en descargas, se le da permisos al script y se ejecuta.

Una vez iniciado sesion con administrador las paginas html a revisar serian: users_management.html, vulnerabilities.html, vulnerabilities_management y el dashboard.html principal que tendria la funcion de un panel de domotica.
